SurMantra 🎵 - Personal Music Portfolio

🌟 About the Project
SurMantra is a personal music portfolio built with Django, HTML, CSS, and JavaScript, designed to showcase my music, biography, and performances. It includes an interactive booking system for gigs and a contact page for inquiries.


⚙️ Tech Stack
Backend: Django (Python)

Frontend: HTML, CSS, JavaScript

Database: SQLite (can be upgraded to PostgreSQL)

Version Control: Git & GitHub
